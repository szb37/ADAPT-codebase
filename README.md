This repo is a simulation based power calculation for the ADAPT study.
See in /notebooks/data_generation.ipynb for the data generation and /notebooks/power_plots.ipynb for the power calculation and associated plots.

If you want to run the data generation you also need R 4.4.2 and BI 1.2 
(https://cran.r-project.org/web/packages/BI/BI.pdf) installed and on the path, all other packages are in the conda environment.
